广川，是集电商、软件开发和大数据平台研发的互联网企业，同时也是一家面向全社会提供IT技术解决方案与信息化服务的企业，致力于通过技术创新，推动社会的发展与变革，为用户和企业创造新的生活方式，为社会创造新的价值。公司创立于2011年，目前拥有60多名伙伴，拥有自身研发的大数据平台，大学生创业平台，同时拥有多项软件著作权和发明专利，是天津市具有影响力的中小型科技企业。
