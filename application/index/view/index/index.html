广川，是集电商、软件开发和大数据平台研发的互联网企业，同时也是一家面向全社会提供IT技术解决方案与信息化服务的企业，致力于通过技术创新，推动社会的发展与变革，为用户和企业创造新的生活方式，为社会创造新的价值。公司创立于2011年，目前拥有60多名伙伴，拥有自身研发的大数据平台，大学生创业平台，同时拥有多项软件著作权和发明专利，是天津市具有影响力的中小型科技企业。

从创立之初，广川便将“为用户提供更好的产品和享受更好的服务”作为自己的使命。成立5年以来，公司秉承“一切以客户的价值为中心，维护用户的切身利益，构建未来息息相关的生态系统”的经营理念，不断坚持完善用户体验，不断坚持技术创新，始终处于上升发展的状态。我们致力于为用户提供“简单，便捷，高效，信赖”的互联网产品、IT产品及服务。因此，广川得到政府的支持，同时与腾讯公司、百度公司保持着良好的合作关系，共同打造具有影响力的软件产品。同时，广川更是与天津大学、南开大学等7所高校建立战略合作体系和研发体系，以及人才培训合作计划。
